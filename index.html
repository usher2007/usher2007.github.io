但是还没结束，怎么修改Makefile可以使得-l语句默认放到所生成的cc语句的最后呢？一番搜索，在Stackoverflow上找到了这篇文章，按照其中所说，把LDFLAGS中的-l部分拆分到LDLIBS中，得到如下Makefile：

此时再调用make，就可以自动的把-l参数放到最后了。

知道了怎么做，还应该知道为什么。
其实man cc看一下文档就会明白了。在 -llibrary节有这么一段话：

It makes a difference where in the command you write this option; the linker searches and processes libraries and object files in the order they are specified. Thus, foo.o -lz bar.o searches library z after file foo.o but before bar.o. If bar.o refers to functions in z, those functions may not be loaded.

也就是说，当linker遇到.o文件中一个无法识别的名字时，就会去按照cc语句中的顺序去搜索库。由于处理到bar.o的时候，-lz在它的左边，所以将不会被搜索到。因此如果在bar.o中使用了z库中的变量或函数，就会无法解析。因此，一般写cc语句的时候，会把-l选项放在所有.o文件之后。
事实上，-l选项一般也的确是放在LDLIBS中的。不知为何，Learn C The Hard Way的作者将其放在了LDFLAGS里。
那么对于object files中的变量，最开始也是类似库文件的顺序查找，但是现代编译器一般都做了优化，会在所有指定的.o文件中进行变量名的查找，所以.o文件的次序变得不那么重要。（参见这篇文章）。

编译成功后，当我第一次调用生成的可执行文件时，有这样的错误信息输出：

./devpkg: error while loading shared libraries: libapr-1.so.0: cannot open shared object file: No such file or directory
这个问题只需要将/usr/local/apr/lib加到/etc/ld.so.conf.d/下的任意文件中（一般会自己新建一个conf文件），然后调用ldconfig即可解决。

说说婚礼

五一回老家办了婚礼，由于我和我老婆两人来自同一个县城，所以时间上倒也没什么耗费，只请了一天年假就搞定了。小县城亲戚多朋友多习俗多，大家对于我们这种三四天搞定的婚礼十分不解。然而谁又知道，我们从一开始就认为这是一场为家里人而办的婚礼呢。

在我的理解中，婚礼，首先是一个仪式，它是一男一女结为夫妻的见证。从这一点上说，婚礼和去民政局领证，并无本质区别。然而，在筹备婚礼的过程中，我发现当代中国的婚礼，仪式感并不是最主要的——各种杂七杂八的风俗，并不是为了营造神圣的氛围，而是为了折腾两位新人。这一点有点像军营中，新兵蛋子一定要被老兵欺负，这样过后回想起来，一届入伍的士兵们才有“兄弟情”。或许这种风俗本无恶意，也仅仅是为了让新人对“结婚”印象深刻罢了。

谈谈国际化

最近高晓松在晓松奇谈里谈到了韩国，不无艳羡的称赞韩国娱乐业中有大量从业人员留学欧美，具有“国际视野”，从而造就了韩国娱乐业的空前繁荣。

我对于“国际视野”也有一些自己的想法。

与具有国际视野相反的，我们经常说某个人山炮、土鳖。到底是什么样的行为、形象，让我们能够区分一个是否具有国际视野呢？是他的穿着打扮？能说一口流利的英语？手机里存着的世界各地的旅行照片？

好像都不是。我15岁从县城到省城上高中，那时我开始觉得县城的人“土”；18岁从省城到北京上大学，又开始觉得省城的人“土”——也并不是城市现代化的高楼大厦让我有如此看法：刚上大学那会儿，五道口也就是一个大城乡结合部，还比不上高中所在的坞城路的建筑洋气。后来我工作了，陆续有县城的亲戚到广州来游玩、探望我。相处的时间长了，我才发现其中的问题在哪里。

2014年下半年书单

工作之后就很难找出整块的时间读书了，即使有整块的时间可以用来阅读，心境也比上学的时候浮躁不少，遇到略有晦涩的部分就想跳过或者直接把书撂下，所以这半年读书的效率大打了折扣。

##非技术类：

  • 《红楼梦》
    • 熟悉我的人都知道我是个坚定的红楼梦黑，红学黑，可是我自己之前却只看过红楼的前二三十回，因为实在是看不下去。这半年我硬着头皮把一百二十回的红楼梦完整的读了一遍，对它的评价由之前的一到两星升到四星左右。我发现随着年岁的增长（上一次读红楼梦还是在初中），我渐渐的能领悟到小说中的家长里短，人情世故，也能觉察到从始至终曹雪芹文字中的一丝丝忧伤和无奈。那扣掉的一星，是由于我认为那些研（chi1）究（si3）红（ren2）学（fan4）的所谓学者们，过分解读了太多的东西，让这部小说承载了太多的东西，使它在普通人眼中失去了小说本来的味道。或许，十年后，我再读红楼梦，会有更深的感悟。
